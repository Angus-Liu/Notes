|  存储引擎   |  事务  |        锁粒度        |            主要应用            |           忌用           |
| :---------: | :----: | :------------------: | :----------------------------: | :----------------------: |
|   MyISAM    | 不支持 | 支持并发操作的表级锁 |         SELECT、INSERT         |       读写操作频繁       |
| MRG_MYISAM  | 不支持 | 支持并发插入的表级锁 |       分段归档、数据仓库       |    全局查找过多的场景    |
|   Innodb    |  支持  |   支持MVCC的行级锁   |            事务处理            |            无            |
|   Archive   | 不支持 |        行级锁        | 日志记录、只支持insert、select | 需要随机读取，更新，删除 |
| Ndb cluster |  支持  |        行级锁        |            高可用性            |        大部分应用        |
