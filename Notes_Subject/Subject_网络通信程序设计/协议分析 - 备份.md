#### 客户端

- 用户注册

  ```
  {
      "type":"SignUp",
      "fromUser":{
          "nickname":"Angus Liu",
          "account":"angus.liu96@gmail.com",
          "password":"123456"
          ... // 可扩展
      }
  }
  ```

- 用户登录

  ```
  {
      "type":"SignIn",
      "fromUser":{
          "account":"angus.liu96@gmail.com",
          "password":"123456"
      }
  }
  ```

- 用户登出

  ```
  {
      "type":"SignOut",
      "fromUser":{
          "account":"angus.liu96@gmail.com"
      }
      "token":"xxx" // 暂时不考虑
  }
  ```

- 私聊消息

  ```
  {
      "type":"PrivateChat",
      "fromUser":{
          "account":"angus.liu96@gmail.com"
      }
      "toUser":{
          "account":"someone@example.com"
      }
      "data":{
          "date":"2018-05-10 17:20"
          "type":"Text", // 后期可扩展为img、voice、video等，暂时不考虑
          "content":"How are you?"
      }
  }
  ```

- 群聊消息

  ```
  {
      "type":"GroupChat",
      "fromUser":{
          "account":"angus.liu96@gmail.com"
      }
      "data":{
          "date":"2018-05-10 17:20"
          "type":"text", // 后期可扩展为img、voice、video等
          "content":"Hello, everyone!"
      }
  }
  ```

- 获取在线用户列表

  ```
  {
      "type":"getUsers",
      "fromUser":{
          "account":"angus.liu96@gmail.com"
      }
  }
  ```

#### 服务端

+ 处理结果

  ```
  {
      "type":"Result",
      "result":true/false,
      "reason":""
  }
  ```

+ 登录成功，返回用户信息（与下同）

  ```
  {
      "type":"UserInfo",
      "user":{
          "nickname":"xxx",
          "account":"1111111@example.com",
          ...
      }
  }
  ```

+ 返回在线用户列表

  ```
  {
      "type":"UserList",
      "toUser":{
          "account":"xxxxxxx@example.com"
      }
      "userList":{
          {"nickname":"xxx","account":"1111111@example.com"},
          {"nickname":"yyy","account":"2222222@example.com"},
          {"nickname":"zzz","account":"3333333@example.com"},
          ...
      }
  }
  ```

+ 通知

  ```
  {
      "type":"",
      "data":{
          "content":"..."
      }    
  }
  ```
