#### 客户端

- 用户注册

  ```
  {
      "type":"SIGN_UP",
      "user":{
          "nickname":"Angus Liu",
          "account":"angus.liu96@gmail.com",
          "password":"123456"
          ... // 可扩展
      }
  }
  ```

- 用户登录

  ```
  {
      "type":"SIGN_IN",
      "user":{
          "account":"angus.liu96@gmail.com",
          "password":"123456"
      }
  }
  ```

- 用户登出

  ```
  {
      "type":"SIGN_OUT", 
      "from":"angus.liu96@gmail.com", // 暂时不考虑
      "token":"xxx" // 暂时不考虑
  }
  ```

- 私聊消息

  ```
  {
      "type":"PRIVATE_CHAT",
      "from":"angus.liu96@gmail.com",
      "to":"someone@example.com",
      "content":{
          "date":"2018-05-10 17:20"
          "type":"Text", // 后期可扩展为img、voice、video等，暂时不考虑
          "info":"How are you?"
      }
  }
  ```

- 群聊消息

  ```
  {
      "type":"GROUP_CHAT",
      "from":"angus.liu96@gmail.com",
      "content":{
          "date":"2018-05-10 17:20"
          "type":"text", // 后期可扩展为img、voice、video等
          "info":"Hello, everyone!"
      }
  }
  ```

- 获取单用户信息

  ```
  {
      "type":"GET_USER_INFO",
      "user":{
          "account":"angus.liu96@gmail.com"
      }
  }
  ```

- 获取在线用户列表

  ```
  {
      "type":"GET_USER_LIST",
      "from":"angus.liu96@gmail.com"
  }
  ```

#### 服务端

+ 处理结果

  ```
  // 注册处理结果
  {
      "type":"SIGN_UP_RESULT",
      "result":true/false,
      "reason":"xxx"
  }
  
  // 登录处理结果
  {
      "type":"SIGN_IN_RESULT",
      "result":true/false,
      "reason":"xxx",
      "user":{ // 登录成功
          "nickname":"xxx",
          "account":"1111111@example.com",
          ...
      }
  }
  
  // 其他处理结果
  {
      "type":"HANDLE_RESULT",
      "result":true,
  }
  ```

+ 返回单用户信息

  ```
  {
      "type":"USER_INFO",
      "user":{
          "nickname":"xxx",
          "account":"1111111@example.com",
          ...
      }
  }
  ```

+ 返回用户列表

  ```
  {
      "type":"USER_LIST",
      "userList":{
          {"nickname":"xxx","account":"1111111@example.com"},
          {"nickname":"yyy","account":"2222222@example.com"},
          {"nickname":"zzz","account":"3333333@example.com"},
          ...
      }
  }
  ```

+ 通知

  ```
  {
      "type":"NOTIFACATION",
      "content":{
          "content":"..."
      }    
  }
  ```
