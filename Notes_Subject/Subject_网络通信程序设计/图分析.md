#### 用户操作

##### 账户

```sequence
participant client as c
participant server as s
participant database as db

note over c:Connect Config

note right of c:User SignUp
c-->s:Type:SignUp
s-->db:User Info
note over db:Add user info\ninto database
db->s:Result
s->c:Type:Result

note right of c:User SignIn
c-->s:Type:SignIn
# note over s:处理client连接
note over s:Add user into\n user list
s-->db:Clien Info
db->s:Result
s->c:Type:Result
note over c:Sign In

note right of c:User SignOut
c-->s:Type:SignOut
note over s:Remove user from\n user list
note over c:Sign Out
```

##### 私聊
```sequence
participant client1 as c1
participant server as s
participant client2 as c2

note right of c1:Query UserList
c1-->s:Type:User/Query
s->c1:Type:User/ClientList

note right of c1:Add User
c1-->s:Type:User/Add
s-->c2:Type:User/Add

note right of c1:Send Private Msg
c1-->s:Type:Chat/Private
s-->c2:Type:Chat/Private

note right of c1:Receive Private Msg
c2->s:Type:Chat/Private
s->c1:Type:Chat/Private

note right of c1:Dele User
c1-->s:Type:User/Dele
s-->c2:Type:User/Dele


```

##### 群聊
```sequence
participant client as c
participant server as s

note right of c:Join Group
c-->s:Type:Group/Join
note over s:Add client\n into group

note right of c:Send Group Msg
c-->s:Type:Chat/Group
note over s:Forward message\n to group

note right of c:Receive Group Msg
s->c:Type:Chat/Group

note right of c:Quit Group
c-->s:Type:Group/Quit
note over s:Remove client\n from group
```
